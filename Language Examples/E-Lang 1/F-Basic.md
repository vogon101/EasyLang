#Basic examples of F-BASIC (Simple basic clone)
##Goal 1 - VERY Simple
```
PRINT "Hello World"
PRINT "Please enter your name below"
IN name
PRINT "Your name is"
PRINT name
```
##Goal 2 - Simple yet usable
```
IN name
PRINT "Your name is " + name
```
##Goal 3 - Variable assignment
```
name = "Freddie"
PRINT "My name is " + name
```
